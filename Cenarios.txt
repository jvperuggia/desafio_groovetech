DESCRIÇÃO: Realizar Cadastro Com Sucesso
NO PAPEL DE usuário  
EU QUERO/PRECISO cadastrar uma conta 
PARA  fazer login. 

Cadastro com Sucesso
Deve ser informado um email válido E ainda não cadastrado 
Os campos indicados como obrigatórios devem ser preenchidos com dados válidos
No final do cadastro deve ser verificado se a mensagem de cadastro com sucesso foi exibida



DESCRIÇÃO: Realizar Cadastro - Email Já cadastrado

NO PAPEL DE usuário  
EU QUERO/PRECISO cadastrar uma conta informando um email já cadastrado
PARA  validar que não é possível realizar cadastro com email repetido. 

Deve ser informado um email já cadastrado 






DESCRIÇÃO: Realizar Cadastro - Email Inválido

NO PAPEL DE usuário  
EU QUERO/PRECISO cadastrar uma conta informando um email inválido
PARA  validar que não é possível realizar cadastro com email inválido. 

Caso seja informado um email inválido o sistema deve exibir uma mensagem informando.
